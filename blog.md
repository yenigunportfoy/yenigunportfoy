# Getting Started with Python

Python is one of the most beginner-friendly languages out there. Here's a simple example:

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

Key reasons to learn Python:
- Simple, readable syntax
- Huge ecosystem of libraries
- Great for web, data science, and automation

# The Art of Writing Good Code

Good code is like good prose — it should be clear, concise, and purposeful. Here are some principles to live by:

1. **Readability** — Code is read far more often than it is written.
2. **Simplicity** — Prefer simple solutions over clever ones.
3. **Consistency** — Follow conventions. Use a linter.

> "Any fool can write code that a computer can understand. Good programmers write code that humans can understand." — Martin Fowler

# Thoughts on Open Source

Contributing to open source is one of the best ways to grow as a developer. You learn from others, build a portfolio, and give back to the community.

Start small — fix a typo in documentation, report a bug, or answer a question in an issue thread. Every contribution counts.
